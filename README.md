

Building/development
--------------------

Prereqs:

* Node
* npm
* [gulp.js](http://gulpjs.com/)

To install:

1. Check out the repo.
2. `npm install` the dependencies.
3. Run `gulp` to start a local server.
4. Hey, check it out, you're running on `http://localhost:8080`
